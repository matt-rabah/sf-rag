# Prompt Builder Key Concepts

## Summary

Prompt Builder key concepts explain the vocabulary and building blocks used to create grounded prompt templates in Salesforce.

These concepts include prompt templates, merge fields, resources, CRM data grounding, and template types such as flex prompt templates.

## Key Concepts

- Prompt Builder
- Prompt template
- Prompt template type
- Flex prompt template
- Merge fields
- Record merge fields
- Flow merge fields
- Apex merge fields
- CRM data
- Resources
- Grounding
- Template workspace
- Generative AI

## Source Content

Prompt Builder uses prompt templates to create reusable prompts for generative AI use cases.

Prompt templates can be grounded with Salesforce CRM data.

Merge fields are used to insert Salesforce data and other resources into prompt templates.

Prompt Builder supports resources such as record fields, flows, related lists, and Apex.

Flex prompt templates can be used when a prompt needs flexible inputs or a reusable prompt structure.

Record merge fields, flow merge fields, and Apex merge fields can be added to prompt templates to provide context to the model.
