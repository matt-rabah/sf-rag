# Explore the New Agentforce Builder

## Summary

The new Agentforce Builder reimagines the agent-building experience to help you create AI agents that are more predictable and reliable.

Its highlights include more determinism from an upgraded Atlas reasoning engine, programmatic control through Agent Script, built-in assistance, an improved user interface, and powerful agent previews.

## Source Content

The new Agentforce Builder reimagines the agent-building experience from the ground up, and it helps you create AI agents that are more predictable and reliable. The new builder has a new home in Agentforce Studio, which is the central hub for creating, customizing, testing, and monitoring AI agents.

More determinism: The Atlas reasoning engine was upgraded to give you more control over how your AI agents behave. Previously, all agent behavior was subject to interpretation by a large language model. The graph-based engine is more structured, so the agent can follow a defined, predictable path when needed.

Programmatic control: Agent Script is the new language for building agents in Agentforce Builder, and it combines the flexibility of natural language with the reliability of programmatic expressions. You can either build in natural language in Canvas view, or toggle to Script view to access the scripting language with syntax highlighting and real-time validation.

Built-in assistance: You can build your agent conversationally with the help of Agentforce embedded right into the user interface. You tell Agentforce what you want your agent to do, and it converts your request directly into Agent Script on your behalf, so you can create, test, troubleshoot, and refine with help from Agentforce.

Improved user interface: Navigating your agent is simpler, flatter, and more transparent. The new builder has a file-based explorer, so there's less digging and searching for the information you need, and it's easier to copy, paste, and reuse an agent's metadata.

Powerful agent previews: When you simulate and test an agent conversation, you also get detailed tracing and debugging. You can view interaction summaries and dig into the steps the agent performed.
