# Toxicity Detection

## Summary

Einstein Trust Layer uses toxicity detection to identify and flag potentially toxic content in prompts and responses.

Toxicity detection supports AI safety by helping detect harmful content in generative AI interactions.

## Source Content

Einstein Trust Layer uses machine learning models to identify and flag toxic content in prompts and responses.

Toxicity detection can apply to prompts.

Toxicity detection can apply to responses.

Toxicity detection helps detect potentially harmful content in generative AI interactions.

Toxicity scores are trust signals that can support governance and observability.

Toxicity scores can be logged as part of the Einstein Trust Layer audit trail.
