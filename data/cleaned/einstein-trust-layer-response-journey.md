# Einstein Trust Layer: Response Journey

## Summary

The Einstein Trust Layer response journey explains how Salesforce handles prompts, masking, toxicity scoring, LLM responses, demasking, audit data, and feedback data.

Audit trail data helps administrators observe Trust Layer behavior and review how prompts and generated responses were processed.

## Source Content

Audit Trail includes the original prompt.

Audit Trail includes the masked prompt.

Audit Trail includes scores logged during toxicity detection.

Audit Trail includes the original output from the LLM.

Audit Trail includes the demasked output.

Audit and feedback data are stored in the customer's instance of Data 360.

Customers control how long audit and feedback data is stored in their instance of Data 360.

Salesforce stores audit and feedback data for 30 days for compliance purposes.

The response journey helps explain how the Einstein Trust Layer supports governance and observability for generative AI interactions.
