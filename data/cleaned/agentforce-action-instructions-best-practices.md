# Best Practices for Agent Action Instructions

## Summary

Action instructions tell an agent what an action does and when to use it.

Good action instructions help the agent select and use the correct action during reasoning.

## Source Content

Action instructions tell an agent what an action does and when to use it.

When creating a custom action, carefully plan and test the instructions.

Good action instructions help the agent understand the action's purpose.

Good action instructions help the agent understand when the action should be selected.

Action instructions are important because the agent uses them during reasoning to decide whether and how to use an action.
