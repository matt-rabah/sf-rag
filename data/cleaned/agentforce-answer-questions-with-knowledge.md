# Agentforce Platform | Answer Questions with Knowledge

## Summary

Answer Questions with Knowledge is an Agentforce action that answers a user's question based on information from relevant knowledge articles and uploaded files.

## Key Concepts

- Agentforce action
- Answer Questions with Knowledge
- Knowledge articles
- Uploaded files
- User questions
- Relevant information
- Agentforce Data Library
- Search index
- Retriever
- Grounded responses

## Source Content

Answer Questions with Knowledge answers a question from a user based on information from relevant knowledge articles and uploaded files.

When an Agentforce Data Library is integrated with AI agents, the data library's search index and retriever are used by the Answer Questions with Knowledge action.

This connects Data 360 grounding and retrieval to the agent action that answers user questions.
