# Ground Prompt Templates with Salesforce Resources

## Summary

Prompt Builder can dynamically ground prompt templates with CRM data using merge fields.

Merge fields can reference Salesforce resources such as record fields, flows, related lists, and Apex.

## Source Content

With Prompt Builder, prompt templates can be dynamically grounded with CRM data using merge fields.

Merge fields can reference Salesforce resources that provide context to the prompt template.

Salesforce resources for prompt grounding include record fields, flows, related lists, and Apex.

Grounding prompt templates with Salesforce resources helps provide relevant context to the model.

Prompt grounding helps generative AI output use Salesforce data instead of relying only on general model knowledge.
