# Prompt Template Types

## Summary

Prompt Builder supports different prompt template types for different generative AI use cases.

Template types help determine how a prompt template is used, what context it receives, and where generated output appears in the Salesforce workflow.

## Source Content

Prompt template types help create prompt templates for particular use cases.

A Sales Email prompt template helps sales teams draft personalized emails for contacts or leads.

A Field Generation prompt template brings generative AI-assisted workflows to fields within Lightning record pages.

A Record Summary prompt template summarizes a record.

A Flex prompt template lets you define your own resources.

Grounded Service Replies support service reply use cases grounded in relevant information.

Different prompt template types support different workflows and use cases.
