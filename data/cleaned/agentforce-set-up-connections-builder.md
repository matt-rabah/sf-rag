# Set Up Connections in Agentforce Builder

## Summary

Connections prepare an Agentforce agent for deployment to channels. A connection provides channel-specific instructions, adaptive response formats, and settings such as the Omni-Channel flows that route conversations.

## Key Concepts

- Agentforce Builder
- Connections
- Channels
- Adaptive response formats
- Omni-Channel flows
- Agentforce Studio
- Explorer panel
- Connected-channel testing

## Source Content

Before adding a connection, create the agent and enable the features required by the intended channel.

In the Agentforce Studio app, open the agent from the Agents tab. In the Explorer panel, select Add Connection. In the Add Connections window, select one or more available connections and add them to the agent.

Only connections available for that agent appear. Select each connection in the Explorer panel to complete its setup.

A connection includes behind-the-scenes instructions that help the agent reason and respond for one or more channels. It can also contain adaptive response formats for structuring responses and delivering content such as images, buttons, links, and videos. Connection settings can include the Omni-Channel flows that route conversations to and from the agent.

After completing a connection's setup, test it by chatting with the agent on the connected channel.
