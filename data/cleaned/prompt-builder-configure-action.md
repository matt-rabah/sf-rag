# Configure an Action in a Prompt Template

## Summary

Prompt Builder can configure actions in a prompt template so the prompt can use existing Salesforce functionality.

Actions such as Flows, Apex, and Retrievers can be added directly to a prompt.

## Source Content

Prompt Builder lets users configure actions in a prompt template.

Existing actions can be modified in Prompt Builder and added directly to a prompt.

Actions that can be configured in a prompt template include Flows, Apex, and Retrievers.

Configuring actions in a prompt template helps the prompt use Salesforce functionality and retrieved context during prompt execution.
