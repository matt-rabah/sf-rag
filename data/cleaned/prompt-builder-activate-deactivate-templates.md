# Activate and Deactivate Prompt Templates

## Summary

Activating a prompt template makes a specific version of the prompt template available to users.

Only one version of a prompt template can be active at a time.

## Source Content

When you activate a prompt template, you make a specific version of the prompt template available to users.

Only one version of a prompt template can be active at a time.

Use activation and deactivation to control which prompt template version users can access.

Prompt template activation should happen after creating, testing, and revising the prompt template.

Deactivation removes a prompt template version from user availability.
