# Use Multiple Versions of a Prompt Template

## Summary

Prompt Builder supports multiple versions of a prompt template so builders can compare prompt constructions and results.

Versioning helps builders safely iterate on prompt templates instead of changing a working prompt without comparison.

## Source Content

Prompt Builder lets builders create and use multiple versions of prompt templates.

Multiple versions can be used to compare prompt constructions.

Multiple versions can be used to compare prompt results.

Prompt template versioning supports safe iteration while developing and improving prompts.

Versioning helps builders test and revise prompt templates before deciding which version should be used.
