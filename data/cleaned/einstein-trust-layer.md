# Einstein Trust Layer

## Summary

The Einstein Trust Layer provides security, privacy, safety, and audit capabilities for generative AI use in Salesforce.

It helps protect sensitive data, score content for toxicity, and support auditing through data stored in Data 360.

## Source Content

Sensitive data is detected and masked before a prompt is sent to the large language model.

Data masking can be configured based on organizational privacy and security requirements.

Einstein Trust Layer scores content for toxicity.

Toxicity scores are logged and stored in Data 360 as part of the audit trail.

Generative AI audit data includes data about Einstein Trust Layer features such as data masking and toxicity scores.

Audit trail data helps track Einstein Trust Layer functionality for governance, observability, compliance, and monitoring.
