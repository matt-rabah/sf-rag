# Create a Hybrid Search Index with Advanced Setup

## Summary

A hybrid search index in Data 360 provides relevant information to generative AI applications by using semantically related chunks, searchable vectors, a vector index, and a keyword index.

- Search index

- Vector index
- Keyword index
- Generative AI grounding

## Source Content

A hybrid search index can be configured for a data model object or an unstructured data model object to provide relevant information to generative AI applications.

Data 360 breaks up the referenced data into semantically related chunks.

Data 360 generates searchable vectors.

Data 360 generates a vector index and a keyword index.

The hybrid search index supports retrieval for grounded generative AI use cases.
