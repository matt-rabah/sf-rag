# Manage Employee Agent Access

## Summary

Users get access to an Agentforce Employee agent by adding the agent to a permission set or profile in Setup.

Agent access controls which users can access and interact with an employee agent.

## Source Content

To give users access to an Agentforce Employee agent created in the new Agentforce Builder, add the agent to a permission set or profile in Setup.

Agent access determines which users can access and interact with the employee agent.

In the legacy Agentforce Builder, access can be managed on the Agent Access tab or by adding agents to a permission set or profile.

Managing employee agent access helps control which users can use the agent.
