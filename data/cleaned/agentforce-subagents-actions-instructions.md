# Subagents, Actions, and Instructions in Agentforce

## Summary

Subagents organize what an agent can handle. A subagent contains actions and instructions.

Actions are the tools an agent can use to do its job. Instructions tell the agent how to make decisions.

## Source Content

In Agentforce, subagents organize the work an AI agent can handle.

A subagent contains actions and instructions.

Actions are the tools available for the job.

Instructions tell the agent how to make decisions.

Salesforce previously used the term topics. Beginning in April 2026, agent topics are called subagents. Salesforce states that there is no functionality change, but documentation may show both terms during the transition.
