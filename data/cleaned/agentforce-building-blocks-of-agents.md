## Source Content

The key components of Agentforce agents include subagents and actions, data, connections and channels, and the reasoning engine.

Subagents and actions help organize what an agent can do.

Data provides the information the agent can use to ground responses and perform work.

Connections and channels determine how the agent connects to users, employees, customers, and systems.

The reasoning engine helps the agent determine how to respond, which subagent or action to use, and how to complete a task.

These building blocks work together to define agent behavior, capability, access, and execution.
