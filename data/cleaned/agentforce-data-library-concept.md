# What Are Data Libraries?

## Summary

Agentforce Data Libraries enhance the accuracy of AI features like Agentforce Agents by connecting them to trusted data sources.

## Key Concepts

- Agentforce Data Libraries
- Trusted data sources
- Agent grounding
- AI response accuracy
- Data 360
- Search indexes
- Retrievers

## Source Content

Agentforce Data Libraries enhance the accuracy of AI features like Agentforce Agents by connecting them to trusted data sources.

Data libraries help agents use approved business information instead of relying only on the model's general knowledge.

A data library can be used to ground an agent's responses in selected source content.
