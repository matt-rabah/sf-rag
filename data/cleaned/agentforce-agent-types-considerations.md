# Agent Types and Considerations

## Summary

Agentforce agent types help builders choose the right kind of agent for a cloud, channel, or common business use case.

Agent types connect to default templates and have their own setup considerations, limitations, permissions, and usage requirements.

## Source Content

Agent types help organize Agentforce agents around clouds and common use cases.

Agent types can include default templates for specific clouds and common business scenarios.

Agentforce Service Agent has its own supported functionality, usage considerations, limitations, allowances, and limits.

Agentforce Employee Agent has its own setup considerations.

Agentforce Default has its own considerations for supported functionality, usage, limits, allowances, and more.

Agent type selection matters because the selected type can affect the templates, permissions, channels, setup requirements, and limitations that apply to the agent.
