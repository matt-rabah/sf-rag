# Create Variables in Agentforce Builder

## Summary

Variables in Agentforce Builder help control how an agent uses action outputs and action inputs.

A variable can be created from an action output and assigned to an action input.

## Key Concepts

- Agentforce Builder
- Variables
- Action output
- Action input
- Subagent

- Agent populated input
- Input assignment
- Output mapping
- Agent control

## Source Content

To create a variable that is set to an action output, select an action in the Actions Available For Reasoning section of a subagent.

Next to an action output, select the option to create a variable.

By default, action inputs are agent populated.

To assign a specific variable to an action input, select an action in the Actions Available For Reasoning section of a subagent, and then select the variable for the action input.

Variables help control how an agent uses action outputs and fills action inputs.
