# Create a Filter to Control Access to Subagents and Actions

## Summary

Filters control when an Agentforce agent can use specific subagents and actions during a conversation.

Filters can limit access based on variables.

## Source Content

Filters can control access to subagents and actions.

Filters make sure an agent can use a subagent or action in a conversation only when certain conditions are met.

Filters can limit access based on variables.

For agents on customer channels, Salesforce recommends using filters to require authentication for subagents and actions that perform tasks for a user.

Filters help control agent behavior by making subagents and actions available only in the right conditions.
