# Agent Actions

## Summary

Agent actions define what an agent can do.

Salesforce provides standard actions for common business use cases, and you can create custom actions for business-specific needs.

## Source Content

Agent actions define the work an agent can perform.

Salesforce provides standard actions that can be added to an agent to handle common business use cases quickly.

Custom actions can be created for business-specific needs.

Actions can be added to subagents from the actions library.

When you add a standard action to an agent, the action can be customized for the use case, including the underlying flow, prompt template, or other reference action.
