# Salesforce Spring '26 Release Notes Index

## Summary

The Salesforce Spring '26 release notes index is used as a release-currency control source.

This source helps identify when a topic may be release-sensitive and should be checked against Spring '26 documentation before relying on older Winter '25 or earlier guidance.

## Key Concepts

- Spring '26 release notes
- Release readiness
- Release note changes
- Monthly feature releases
- Data 360 release notes
- Agentforce Platform release notes
- Feature availability
- Release updates
- Known issues

## Source Content

The Spring '26 release notes provide links to Salesforce release readiness resources, release changes, release updates, and product-specific release note areas.

The release notes include a Release Note Changes page for updates made after the release notes were published.

The release notes include guidance on how and when features become available.

The release notes include a Data 360 section. Data 360 features and changes can be released as often as monthly, so Data 360 topics should be treated as release-sensitive.

The release notes include an Agentforce Platform section for predictive and generative AI updates.

The release notes identify Salesforce Known Issues as a separate source.

## Exam Relevance

This source supports release-currency checks for the Salesforce Certified Agentforce Specialist Spring 2026 exam.

It should not be treated as the primary authority for exam scope. The exam guide remains the scope authority.

Use this source to flag topics that may have changed since Winter '25 or earlier, especially Data 360, Agentforce Platform, release updates, feature availability, and monthly feature changes.

## Notes

Use this source as a freshness and release-control layer.

Do not use this source alone to answer detailed concept questions unless the question is specifically about Spring '26 release readiness, release-note changes, feature availability, monthly releases, or where to check for release-sensitive updates.
