# Create an Answer Questions with Knowledge Prompt Template

## Summary

An Answer Questions with Knowledge prompt template enables agents to answer questions using information from an Agentforce Data Library.

To create and customize an Answer Questions with Knowledge prompt template, first create an Agentforce Data Library.

## Source Content

To create and customize an Answer Questions with Knowledge prompt template, first create an Agentforce Data Library.

The Answer Questions with Knowledge prompt template enables agents to answer questions using grounded information.

Salesforce recommends copying and editing the standard prompt template when customizing the Answer Questions with Knowledge template.

The standard prompt template includes key pieces of a successful prompt, including instructions for the LLM, data resources, and relevant inputs like the user query.

To test the Answer Questions with Knowledge prompt template, use the Query and Retriever ID.

The Query is the question asked by the user.

The retriever is already created and activated in AI Models.

When the template is active, the customized prompt template is used when a user asks an agent a question.
