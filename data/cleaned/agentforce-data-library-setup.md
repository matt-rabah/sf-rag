# Setting Up Data Libraries

## Summary

Agentforce Data Libraries require Data 360 setup. When a data library is configured, the selected source content is indexed in Data Cloud, where a search index and retriever are automatically created.

## Source Content

To use Agentforce Data Libraries, Data 360 must be set up.

When configuring a data library, supported data sources include knowledge articles, file uploads, web search, and custom retrievers.

The selected data source is indexed in Data Cloud.

A search index and retriever are automatically created for the data library.

When integrated with AI agents, the data library's search index and retriever are used by the Answer Questions with Knowledge action.

Data libraries can be created and assigned to features from Agentforce Data Library setup or when configuring an agent in Agentforce Builder.
