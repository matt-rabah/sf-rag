## Source Content

What does an Agentforce agent template include?

An Agentforce agent template includes relevant subagents and actions, default system messages, variables, and filters.

You can create agents in multiple ways.

To start with an agent for a common business use case, create an agent from a template.

To create an agent from a template, you must have the permissions for the associated agent type.

You must also have any additional permissions required for the template.

Templates help builders start from a configured agent structure instead of building every part from scratch.
