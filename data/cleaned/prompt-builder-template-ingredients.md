# Ingredients of a Prompt Template

## Summary

An effective prompt template gives the LLM enough information to generate a personalized response.

A strong prompt template is scalable and reusable across different use cases.

## Source Content

An effective prompt template answers the questions needed for the LLM to generate a personalized response.

A strong prompt template is scalable so that it can be reused across different use cases.

Prompt Builder helps design prompt templates that support reusable generative AI workflows.

Prompt template ingredients help builders structure prompts with enough instruction and context for useful outputs.

Prompt templates should be designed so the LLM understands the task, the expected response, and the relevant context.
