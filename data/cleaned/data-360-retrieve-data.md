# Retrieve Data

## Summary

A retriever is used to search and return relevant data from a search index.

## Key Concepts

- Retriever
- Search index
- Relevant data
- Grounding
- Prompt templates
- Agents

## Source Content

A retriever searches and returns relevant data from a search index.

Retrievers help make source-backed information available to prompts and agents.

A retriever is connected to a search index and controls what information is returned for a particular use case.

Retrievers can be created and customized in AI Models.
