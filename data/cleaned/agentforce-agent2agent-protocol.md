# Agent2Agent Protocol

## Summary

Agent2Agent, or A2A, is an open standard that lets AI agents from different platforms securely communicate and collaborate.

A2A supports multi-agent interoperability by giving specialized agents a consistent way to share context and delegate tasks across systems.

## Source Content

Agent2Agent, or A2A, is an open standard that allows AI agents from different platforms to securely communicate and collaborate.

A2A supports AI agent interoperability.

A2A gives specialized agents a consistent way to share context and delegate tasks across systems.

The Agent2Agent protocol uses a client-server architecture.

In the client-server architecture, a client agent orchestrates work and a remote agent processes tasks through structured, stateful messaging.

A2A helps connect agents across platforms so they can collaborate beyond one system.
