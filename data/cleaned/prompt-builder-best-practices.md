# Best Practices for Building Prompt Templates

## Summary

Prompt templates should be concise, clear, and easy to understand.

Good prompt templates use natural language and avoid unnecessary jargon or overly technical wording.

## Source Content

Prompt templates should be concise and easy to understand.

Avoid using industry jargon or technical terms when simpler natural language is clearer.

Use natural language to help the prompt template communicate clearly with the large language model.

Prompt templates should provide enough instruction and context for the model to generate useful responses.

Prompt templates should be tested and refined to improve output quality.

Clear prompt writing helps users get more reliable generative AI responses.
