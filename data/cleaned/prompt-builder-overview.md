# Prompt Builder

## Summary

Prompt Builder helps create, test, revise, customize, and manage prompt templates.

Prompt templates can incorporate CRM data from merge fields that reference record fields, flows, related lists, and Apex.

## Source Content

Prompt Builder helps users create, test, revise, customize, and manage prompt templates.

Prompt templates can incorporate CRM data from merge fields.

Merge fields can reference record fields, flows, related lists, and Apex.

Prompt Builder helps safely connect users and their data with large language models.

Prompt templates integrate generative AI moments into user workflows.
