# Deploy Your Agent to Channels

## Summary

Deploying an Agentforce agent to channels makes the agent available to customers or employees.

Activation and channel deployment are part of moving an agent from build-and-test work into real user access.

## Source Content

Activate your agent to make it available to customers or employees.

When an agent is deployed to one or more channels, including the Agentforce panel, users can access and interact with the agent through those channels.

Deploying to channels is part of making an agent available outside the builder.

Channel deployment should come after agent configuration, testing, and activation readiness.
