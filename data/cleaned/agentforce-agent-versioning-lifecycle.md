# Versioning and Editing Agents

## Summary

Agent versioning supports the lifecycle of building, modifying, testing, activating, and updating an Agentforce agent.

When you create a new agent or a new agent version, Salesforce creates an editable draft of the agent.

## Source Content

When you create a new agent or a new agent version, you create an editable draft of the agent.

After creating or versioning an agent, modify the agent and test your changes.

After testing changes, activate the agent.

When updates are needed after activation, create a new agent version so changes can be made and tested before activation.

Agent versioning helps separate editable draft changes from activated agent behavior.
