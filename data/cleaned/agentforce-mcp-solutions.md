# MCP Solutions

## Summary

Model Context Protocol, or MCP, helps AI applications connect to external tools, services, and data through MCP servers.

An MCP server securely processes requests on the target platform and returns data or action results to the AI application.

## Source Content

In MCP-enabled AI solutions, a user asks a question or requests an action related to a connected service.

The AI application connects to the appropriate MCP server.

The MCP server securely processes the request on the target platform.

Data or action results return to the AI application.

The AI application displays information or confirms completed actions.

MCP helps AI applications use external tools, services, and data through a standardized server-based pattern.
