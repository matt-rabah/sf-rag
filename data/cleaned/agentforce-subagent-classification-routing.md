# Subagent Classification and Routing

## Summary

Subagent classification is how an Agentforce agent selects the most relevant subagent based on what the user wants to do and the jobs the agent can do.

When an agent is created, Salesforce adds a special subagent called the Agent Router.

## Source Content

When you create an agent, a special subagent called the Agent Router is added to your agent.

The Agent Router is defined as the starting subagent for every agent conversation.

By default, the agent uses the Agent Router for subagent classification.

Subagent classification is the process of selecting the most relevant subagent based on what the user wants to do and the jobs that the agent can do.

Subagent classification helps route the conversation to the correct subagent for the user's request.
