# Create a Flex Prompt Template

## Summary

A Flex prompt template is used when a prompt needs flexible inputs or reusable prompt structure.

In Prompt Builder, a builder creates a new prompt template and selects Flex as the prompt template type.

## Source Content

To create a Flex prompt template, go to Prompt Builder and create a new prompt template.

For Prompt Template Type, select Flex.

Flex prompt templates can use different inputs based on the use case.

Flex templates are useful when a prompt needs flexible inputs or a reusable prompt structure.

After creating the Flex prompt template, use the Prompt Template Workspace to add prompt content, insert resources, preview the resolved prompt, test the output, and activate the template when ready.
