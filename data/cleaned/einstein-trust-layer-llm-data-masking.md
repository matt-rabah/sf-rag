# Large Language Model Data Masking

## Summary

Einstein Trust Layer uses data masking to protect sensitive information before prompts are sent to a large language model.

Data masking helps prevent personally identifiable information, or PII, from being exposed to external LLMs.

## Source Content

Einstein Trust Layer identifies and masks sensitive information in prompts before sending the prompts to the large language model.

Data masking protects sensitive data such as personally identifiable information, or PII.

Data masking helps prevent sensitive information from being exposed to external LLMs.

Einstein Trust Layer data masking can use pattern-based masking.

Einstein Trust Layer data masking can use field-based masking.

LLM data masking is part of the Trust Layer's governance and security controls for generative AI.
