# Preview and Test in Agentforce Builder

## Summary

Preview and Test in Agentforce Builder lets you manually test an agent conversation before relying on the agent in production.

The Preview panel lets you chat with the agent. The Interaction Summary panel shows how the agent handled the interaction.

## Key Concepts

- Agentforce Builder
- Preview panel
- Interaction Summary panel
- Manual testing
- Subagent selection
- Action execution
- Reasoning
- Agent testing
- Debugging
- Conversation trace
- Agentforce Assistant

## Source Content

Manual testing in Agentforce Builder provides a step-by-step trace of an agent's interactions and processes.

Use the Preview panel to chat with the agent.

Use the Interaction Summary panel to review subagent selection, action execution, and reasoning.

You can get more details about steps in the interaction by conversing with the Agentforce Assistant.

Preview testing helps validate whether the agent selected the intended subagent, executed the correct action, and reasoned as expected.
