# Agentforce Analytics Reports

## Summary

Agentforce Analytics uses Data 360 reports to provide insight into agent performance, usage, success, and adoption.

These reports help admins and builders monitor whether agents are being used effectively and whether changes are needed.

## Source Content

Agentforce Analytics is based on Data 360 reports.

Agentforce Analytics helps provide insight into how agents are performing.

Agentforce Analytics reports can show who is using agents.

Agentforce Analytics reports can show how successful agents are.

Agentforce Analytics reports can show agent adoption level.

Agentforce Analytics insights can help determine whether changes are needed to drive adoption.

Agentforce Analytics insights can help determine whether subagents or actions should be adjusted to improve user satisfaction.
