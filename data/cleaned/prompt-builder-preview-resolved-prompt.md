# Preview a Resolved Prompt

## Summary

Previewing a resolved prompt helps test how merge fields and grounding data are applied before the prompt template is sent to the LLM.

Prompt Builder lets you select test data in Preview Settings, display the resolved prompt, and revise the prompt template before sending it to the model.

## Source Content

To preview a resolved prompt, create a prompt template with merge fields.

Use Preview Settings to select test data for merge fields.

Use Display Resolution to view how the prompt resolves.

Each time Preview is clicked, the prompt is displayed with the grounding data selected in Preview Settings.

To ensure the prompt template yields expected results, modify the prompt template and preview again before sending the prompt template with masked data to the LLM.
